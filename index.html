什么是gulp?

gulp是一个前端项目构建工具，通过gulp你可以对你的项目进行编译，并且压缩你的js、css文件和你的图片资源。
下面我们通过一个例子来学习gulp。

1. 安装node.js

。。。

2. 安装Gulp

npm install gulp -g

note: 使用gulp -v命令查看Gulp是否安装成功

3. 新建一个项目

3.1 创建一个文件夹名字叫test，它包含一些子文件夹，参照如下的树形图创建:

  • src（存放预处理的html文件）
    • images（未压缩的图片文件）
    • scripts（多个js文件）
    • styles（多个css文件）
  • build（载入时的文件）
    • images（压缩后的图片文件）
    • scripts（单个js文件）
    • styles（单个css文件）

note：build文件夹中的文件是自动创建出来的

3.2 install gulp

npm install gulp --save-dev

这会在test文件夹下面创建一个node_modules文件夹其中包含了Gulp等依赖

3.3 在test文件夹下面创建一个空的gulpfile.js作为配置文件
